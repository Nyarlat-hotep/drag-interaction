# Dark Mode Toggle Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add a light/dark mode toggle to the drag-interaction app, defaulting to dark mode, matching the condition-builder pattern.

**Architecture:** CSS custom properties on `:root` (dark default) and `:root[data-theme="light"]` (light override). Theme state lives in `App.jsx` as `useState('dark')`, applied via `useEffect` to `document.documentElement.dataset.theme`. Sun/moon SVG toggle button positioned top-right inside the `.app` card.

**Tech Stack:** React, plain CSS custom properties, no external libraries.

### Task 4: Add theme state and toggle button to `App.jsx`

**Files:**
- Modify: `src/App.jsx`

**Step 1: Add theme state and useEffect after existing state declarations**

After line `const physicsRef  = useRef()` (around line 20), add:

```jsx
const [theme, setTheme] = useState('dark')

useEffect(() => {
  document.documentElement.dataset.theme = theme === 'light' ? 'light' : ''
}, [theme])
```

Make sure `useState` and `useEffect` are already imported (they are on line 1).

**Step 2: Add the toggle button inside the `.app` div**

Inside the `return`, immediately after `<div className="app" ...>` opens (before the `<h1>`), add:

